# Tornado_Cimatology
This repository is for my tornado climatatology project for GEO290 Enviromental GIS at Valparaiso University.

Data sources

CAPE: https://psl.noaa.gov/data/gridded/data.20thC_ReanV2c.monolevel.mm.html
